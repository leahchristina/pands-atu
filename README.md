# pands-atu - Programming and Scripting (PANDS)

This repository contains my weekly task submissions for the Programming and Scripting module, part of the Higher Diploma in Data Analytics at ATU.

I recommend using Visual Studio Code to run the notebook.

---

## Repository Purpose

The purpose of this repository is to store and present the weekly tasks completed throughout the semester for assessment.  

---

## Repository Structure

- `pands-weekly-tasks/`  
  Contains all completed weekly task submissions in Python and notebook format.

- `.gitignore`  
  Specifies files and folders excluded from version control.

- `README.md`  
  Overview and explanation of repository contents.
